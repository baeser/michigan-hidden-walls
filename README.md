# Michigan's Hidden Walls — A Raisin in the Sun Lesson

An interactive single-class lesson (45–60 min) for high-school students reading
*A Raisin in the Sun*, using the University of Michigan's **Justice InDeed**
covenant map to compare segregation in the North vs. the Deep South.

**Live lesson:** https://baeser.github.io/michigan-hidden-walls/

Built for **Dexter Community Schools** with local anchors at the Ann Arbor
Country Club and Portage Lake.

## What's here

- `index.html` — the interactive lesson (5 steps, embedded live covenant map)
- `exit_ticket.pdf` — printable one-page exit ticket for students to upload to Canvas
- `make_exit_ticket.py` — reportlab script that generates the PDF (run `python3 make_exit_ticket.py` to regenerate)

## Lesson flow

1. **Opening quick-write** — how does Mr. Lindner keep the Youngers out?
2. **Explore the map** — partner task clicking covenants around the Country Club and Portage Lake
3. **North vs. South** — *de jure* vs. *de facto* segregation
4. **Back to the play** — close reading of Act II, Scene 3
5. **Exit ticket** — written response, turned in via the printable PDF
